# Contributing

Thank you for helping improve Job Search Assistant.

1. Fork the repository.
2. Create a focused branch: `feature/description` or `fix/description`.
3. Add or update tests.
4. Run `python -m unittest discover -s tests -v`.
5. Open a pull request explaining the user impact.

Please avoid features that automate mass applications, bypass access controls, or collect credentials. New integrations must be opt-in and document their privacy implications.
