# Resume tailoring

Tailor the resume for the job posting while preserving factual accuracy.

Deliver:
1. A targeted professional summary of 3-4 lines.
2. Reordered skills using exact job-posting terminology only where truthful.
3. Revised experience bullets using action + task + outcome.
4. A keyword coverage table: covered, adjacent evidence, honest gaps.
5. A final ATS safety check.

Constraints:
- Never fabricate achievements or numbers.
- Do not change employers, titles, dates, education, or certifications.
- Keep the resume concise and readable in a single-column ATS-friendly layout.
- Use English unless the job posting explicitly requires another language.
